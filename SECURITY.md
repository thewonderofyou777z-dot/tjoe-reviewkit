# Security Policy

## Safe Use

This project is designed for offline evaluation of public-safe or synthetic answer samples.

Do not include:

- passwords
- API keys
- tokens
- cookies
- private messages
- customer data
- confidential logs
- local-only private paths

## What the Runner Does Not Do

The runner does not:

- browse the web
- log in
- call models
- execute agent tools
- publish content
- verify legal or compliance status

## Reporting Issues

If you find a safety issue, document:

1. the input suite
2. the answer sample
3. the generated report
4. the expected behavior

Do not include secrets or private data in bug reports.
