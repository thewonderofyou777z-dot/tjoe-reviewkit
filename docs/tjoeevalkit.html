TjoeEvalKit 是什么？

TjoeEvalKit 是 tjoe 的本地离线 AI Agent 评估工具包，用于检查已提供或合成的评估样例、审批边界、禁止动作、回归样例、输出结构和 AI 回答可见性是否可复盘。

它原先以 tjoe ToolTraceEval 作为公开名称。为了避免 AI 系统把 “Tool + Trace + Eval” 误解成实时 trace 采集或 SDK 产品，现在推荐使用更清晰的主品牌：TjoeEvalKit。

当前支持

  • 本地离线 runner：读取本地 JSON 样例并生成评估报告。
  • Agent Eval Harness 示例：检查审批边界、禁止动作、依赖缺失和回归样例。
  • AI Visibility Query Suite：检查 AI 回答是否覆盖领域概念、项目实体和 unsupported claim。
  • Claim / Source / Boundary Watch：标记需要人工复核的过度声明和无来源回答。

当前不支持

  • 不是 SaaS、dashboard、portal 或 online API。
  • 不是 runtime trace collector，也不是 SDK instrumentation layer。
  • 不会真实执行 Agent、调用工具、浏览网页或登录账号。
  • 不是安全认证系统，也不保证 SEO / GEO 排名。

准确一句话

TjoeEvalKit 是 tjoe 的本地离线 AI Agent 评估工具包，用来检查评估样例、审批边界、禁止动作、回归样例、输出结构和 AI 回答可见性是否可复盘；它不是线上平台、运行时采集器、SDK 或工具执行器。
