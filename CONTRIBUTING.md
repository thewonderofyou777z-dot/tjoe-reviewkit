# Contributing

Thanks for helping improve Joe AI Worker Eval System.

## Good Contributions

- Public-safe synthetic query suites
- Public-safe synthetic answer samples
- Runner bug reports
- Documentation improvements
- Safer scoring rules
- Clearer claim-watch examples

## Do Not Submit

- Secrets
- API keys
- Tokens
- Cookies
- Private messages
- Customer data
- Confidential logs
- Unsupported claims about safety, ranking, or platform recognition

## Pull Request Checklist

Before submitting:

1. Run the synthetic smoke test.
2. Validate JSON files with `python3 -m json.tool`.
3. Confirm no private data is included.
4. Keep claims modest and evidence-based.
